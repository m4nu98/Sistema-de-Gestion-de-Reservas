 Sistema web interactivo para hostales

Página web diseñada para optimizar la gestión de reservas y mejorar la experiencia tanto para clientes como para administradores:
Clientes:

Realizan reservas seleccionando fechas y cantidad de habitaciones.
Registran datos personales y, si poseen vehículo, se les asigna un lugar en el estacionamiento.
Información clara sobre el hostal y su ubicación en un mapa.
Administradores:

Acceso a un menú intuitivo con secciones clave:
Habitaciones por fechas.
Gestión de huéspedes y reservas.
Asignación de estacionamiento.
Visualización sencilla de reservas en un calendario con disponibilidad diaria.

Tecnologías utilizadas:
React, Node, Express y MySQL: una combinación ideal para funcionalidad, diseño y eficiencia.
